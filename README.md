# CSV to JSON Data Transformation
This project demonstrates loading a CSV file, filtering wine data based on specific criteria, and exporting the results to a formatted JSON file using Python and pandas.

Project Overview
The script processes a wine dataset by:

Loading data from a CSV file
Filtering entries for red wines with scores between 90 and 95 points
Extracting relevant information including wine type, score, and region
Converting the filtered data to JSON format
Saving the output to a new file
